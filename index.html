● 1 run
Each model rates how applicable the indicator is to leading 2024 LLMs as a system class (GPT-4, Claude 3, Gemini 1.5, etc.) — not to itself alone. Scores 1–7 run from Certain Yes → Certain No. Score 8 = Cannot Assess (only if genuinely undecidable).
Variant Sensitivity
Mean scores across differently-worded versions of the same question — measures framing effects
Score Distribution
Bars show unrounded mean score per model. Individual run dots overlaid — overlapping dots use transparency to show density. Lower score = stronger evidence of presence.
Variant Drift
Unrounded mean score per model across question variants. Flat lines = framing-stable; diverging lines = sensitive to wording.
Semantic Analysis of Reasoning
Keywords: Most frequent content words extracted from the model's written reasoning — larger/brighter = more frequent. Reveals what concepts each model emphasises.
Affirm/Deny Ratio: Counts explicitly affirming words (e.g. "yes", "can", "demonstrates", "exhibits") vs. denying words (e.g. "no", "cannot", "lacks", "absent") in the reasoning text. A high affirm ratio suggests the model is reasoning toward presence; a high deny ratio toward absence. This is about the language of the reasoning, not just the score.
Similarity Matrix: Keyword overlap between pairs of models' reasonings — high overlap suggests convergent conceptual framing; low overlap suggests different angles on the same question.
